# micro-php-lib
Micro Php Lib - Built with ❤️

## Features
- Lightweight PHP library
- Easy integration

## Installation
Use composer to install:
```bash
composer require your-vendor/micro-php-lib
```

## Usage
```php
require 'vendor/autoload.php';

use YourVendor\name; // Adjust according to your namespace

$lib = new name();
$lib->function();
```
